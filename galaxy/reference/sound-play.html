# Play Sound (Deprecated)

Grammar — Play soundLink for players (at volume% volume, skip the first offset seconds)
Flags — Action | Hidden

Plays a sound for the specified player group. Use the “Last Played Sound” function to refer to this sound.

# Arguments

  • soundlink — Sound Link
  • playergroup — Players
  • fixed — Volume
  • fixed — Offset

Returns — void

void SoundPlay(
	soundlink lp_soundLink,
	playergroup lp_players,
	fixed lp_volume,
	fixed lp_offset,
);
