# Set Leaderboard Item Text

Grammar — Set board|Leaderboard item text at column column and row row to text
Flags — Native | Action

Sets the text of a leaderboard item.

# Arguments

  • int [ preset::Board ] — Board
  • int — Column
  • int — Row
  • text — Text

Returns — void

native void BoardItemSetText(
	int inBoard,
	int inCol,
	int inRow,
	text inText,
);
