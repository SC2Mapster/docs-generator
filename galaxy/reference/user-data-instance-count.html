# Number Of User Data Instances

Grammar — Number of user data instances for userType
Flags — Native | Function

Returns the number of instances defined within the given user type. User types are defined within the Data module.

# Arguments

  • string [ gamelink::User ] — User Type

Returns — int

native int UserDataInstanceCount(string inType);

# Related

Category: User Data / Basic

  • [F] — int — Number Of User Data Instances — ( UserDataInstanceCount )
  • [F] — int — Index Of User Data Instance — ( libNtve_gf_UserDataInstanceGetIndex )

  • [F] — int — Number Of User Data Fields — ( UserDataFieldCount )
  • [F] — string [ userfield ] — User Data Field — ( UserDataField )
  • [F] — int [ preset::UserDataType ] — Type Of User Data Field — ( UserDataFieldType )
