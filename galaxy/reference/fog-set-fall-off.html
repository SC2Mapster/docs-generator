# Set Fog Falloff

Grammar — Set fog’s falloff to falloff
Flags — Native | Action

Sets the falloff for environmental fog. Falloff determines how far from the fog start height the environmental fog will extend.

# Arguments

  • fixed — Falloff

Returns — void

native void FogSetFallOff(fixed f);
