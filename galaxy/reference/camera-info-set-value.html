# Set Camera Object Property

Grammar — Set c|Camera Object type to value
Flags — Native | Action

Sets a camera object property.

# Arguments

  • camerainfo — CameraObject
  • int [ preset::CameraObjectPropertyType ] — Property
  • fixed — Value

Returns — void

native void CameraInfoSetValue(
	camerainfo c,
	int type,
	fixed value,
);
