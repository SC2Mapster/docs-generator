# Move Boss Bar

Grammar — Move boss bar bossBarID to anchor of screen with offset (offsetX, offsetY)
Flags — Action

Moves the specified boss bar to the specified coordinates.

# Arguments

  • int — BossBarID
  • int [ preset::Anchor ] — Anchor
  • int — OffsetX
  • int — OffsetY

Returns — void

	int lp_anchor,
	int lp_offsetX,
	int lp_offsetY,
