# Arccosine

Grammar — Acos(x)
Flags — Native | Function

Returns the arccosine of a number. Allowed range is -1 to 1. The returned value is given in degrees.

# Arguments

  • fixed — Value

Returns — fixed

native fixed ACos(fixed x);
