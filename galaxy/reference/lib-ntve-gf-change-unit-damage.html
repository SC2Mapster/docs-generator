# Change Unit Damage

Grammar — Make unit takeDeal option damage
Flags — Action

Modifies the amount of damage a unit deals or takes. Minimal means 10% damage. Mega means 300% damage.

# Arguments

  • unit — Unit
  • int [ preset::UnitDamageTakenDealt ] — Take/Deal
  • int [ preset::UnitDamageChangeOption ] — Option

Returns — void

void libNtve_gf_ChangeUnitDamage(
	unit lp_unit,
	int lp_takeDeal,
	int lp_option,
);
