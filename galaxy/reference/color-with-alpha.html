# Convert Reals To Color With Alpha

Grammar — Color(r%, g%, b%, a%)
Flags — Native | Function

Converts a real value to a color value with alpha settings.

# Arguments

  • fixed — Red
  • fixed — Green
  • fixed — Blue
  • fixed — Alpha

Returns — color

native color ColorWithAlpha(
	fixed r,
	fixed g,
	fixed b,
	fixed a,
);
