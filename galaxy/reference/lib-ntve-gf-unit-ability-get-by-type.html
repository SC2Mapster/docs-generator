# Unit Ability By Class

Grammar — unit ability of class abilityClass at index
Flags — Function

Returns the ability of specified class in the specified unit’s ability array at the specified index.

# Arguments

  • unit — Unit
  • int [ preset::AbilityClass ] — Ability Class
  • int — Index

Returns — string [ gamelink::Abil ]

	unit lp_unit,
	int lp_abilityClass,
	int lp_index,
);
