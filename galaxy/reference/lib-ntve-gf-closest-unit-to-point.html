# Closest Unit To Point Old

Grammar — Closest unit to point in group|Unit Group
Flags — Function | Hidden

Picks the closest unit to the point specified in a unit group.

# Arguments

  • point — Point
  • unitgroup — Group

Returns — unit

	unitgroup lp_group,
