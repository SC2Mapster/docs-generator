# Point From XY

Grammar — Point(x, y)
Flags — Native | Function

Returns a 2D location as a point. This allows you to specify any location on the map grid as a viable value for a point parameter.

# Arguments

  • fixed — X
  • fixed — Y

Returns — point

native point Point(fixed x, fixed y);
