# Add Unit To Control Group

Grammar — Add unit to control group controlGroup|Group Number for player player
Flags — Native | Action

Adds a unit to a control group for a player.

# Arguments

  • int — Player
  • int — Control Group
  • unit — Unit

Returns — void

native void UnitControlGroupAddUnit(
	int inPlayer,
	int inGroup,
	unit inUnit,
);
