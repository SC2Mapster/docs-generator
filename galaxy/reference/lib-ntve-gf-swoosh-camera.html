# Zoom Camera

Grammar — Zoom camera for player player at location from distance1|Distance From to distance2|Distance To over duration seconds
Flags — Action

Zoom the camera in on a location.

# Arguments

  • int — Player
  • fixed — DistanceFrom
  • fixed — DistanceTo
  • point — Location
  • fixed — Duration

Returns — void

void libNtve_gf_SwooshCamera(
	int lp_player,
	fixed lp_distance1,
	fixed lp_distance2,
	point lp_location,
	fixed lp_duration,
);
