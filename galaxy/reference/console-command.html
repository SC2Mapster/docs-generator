# Console Command

Grammar — Run console command: text|Command (defaults|Enable/Disable defaults, macros|Enable/Disable macros)
Flags — Native | Action

Runs a console command.

# Arguments

  • string — Text
  • bool [ preset::Enable_Disable_Option ] — Defaults
  • bool [ preset::Enable_Disable_Option ] — Macros

Returns — bool

native bool ConsoleCommand(
	string inText,
	bool allowDefault,
	bool allowMacros,
);
