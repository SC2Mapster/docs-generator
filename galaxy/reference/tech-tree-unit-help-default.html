# Set Help Panel Default Visibility

Grammar — Set help panel to inShow|Show/Hide units by default for player inPlayer
Flags — Native | Action

Determines whether or not units show up in the help panel by default. Use the “Show/Hide Help For Unit” action to show or hide individual unit types.

# Arguments

  • int — Player
  • bool [ preset::ShowHideOption ] — Show

Returns — void

native void TechTreeUnitHelpDefault(
	int player,
	bool display,
);
