# Declare Next Town

Grammar — Create a new town for player player at point center|center
Flags — Action

Create a new town for player at the specified point.

# Arguments

  • int — Player
  • point — Center

Returns — void

void libNtve_gf_DeclareNextTown(
	int lp_player,
	point lp_center,
);
