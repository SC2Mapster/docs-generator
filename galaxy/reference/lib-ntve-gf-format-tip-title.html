# Format Tip Title

Grammar — Format tip title as type
Flags — Function

Returns the specified text string with some formatting added to it.

# Arguments

  • text — Title
  • int [ preset::TipType ] — Type

Returns — text

text libNtve_gf_FormatTipTitle(text lp_title, int lp_type);
