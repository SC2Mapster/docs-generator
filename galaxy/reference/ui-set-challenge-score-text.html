# Set Challenge Score Text

Grammar — Set the score text of challenge to text for playerGroup
Flags — Native | Action | Restricted

This action will only work for Blizzard maps.

# Arguments

  • playergroup — PlayerGroup
  • string — Challenge
  • text — Text

Returns — void

native void UISetChallengeScoreText(
	playergroup players,
	string challengeId,
	text scoreText,
);
