# Enable Campaign Completed Saves

Grammar — enable Campaign Completed Saves for players
Flags — Native | Action | Restricted

If campaign completed saves are enabled, all saves made will be copied into the file specified by the CompletedSaveName field in Campaign Data in GameUI Data for the current campaign

# Arguments

  • playergroup — Players
  • bool [ preset::Enable_Disable_Option ] — Enable

Returns — void

native void CampaignProgressEnableCampaignCompletedSaves(
	playergroup players,
	bool inDisable,
);
