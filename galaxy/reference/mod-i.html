# Modulo (Integer)

Grammar — x mod y
Flags — Native | Function

Uses integer values. Returns the remainder after the dividend has been divided by the divisor.

# Arguments

  • int — Dividend
  • int — Divisor

Returns — int

native int ModI(int x, int m);
