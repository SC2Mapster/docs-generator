# Cutscene Bookmark Fired

Grammar — Cutscene cutscene fired bookmark bookmark
Flags — Native | Event

When a bookmark is passed in a cutscene, this event is fired so that triggers may respond to it.

# Arguments

  • int [ preset::Cutscene ] — Cutscene
  • string — Bookmark

Returns — void

native void TriggerAddEventCutsceneBookmarkFired(
	trigger t,
	int inCutscene,
	string inBookmarkName,
);
