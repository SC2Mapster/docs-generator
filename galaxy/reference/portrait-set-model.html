# Set Portrait Model

Grammar — Set Portrait model to Model (wait until loaded)
Flags — Native | Action

Sets the model for a portrait.

# Arguments

  • int [ portrait ] — Portrait
  • string [ gamelink::Model ] — Model
  • bool [ preset::Trigger_Wait_Option ] — Wait For Load

Returns — void

	int p,
	string modelLink,
	bool waitUntilLoaded,
