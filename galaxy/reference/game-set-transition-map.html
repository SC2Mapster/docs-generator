# Set Transition Map

Grammar — Set game transition map to transition
Flags — Native | Action

Sets the transition map for transition saves. Only supported in single player offline games.

# Arguments

  • string — Transition

Returns — void

native void GameSetTransitionMap(string transitionMap);
