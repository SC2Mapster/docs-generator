# Create Alert At Point

Grammar — Create a alertType|Alert Type alert at point for player player with the text: text (Custom icon: icon)
Flags — Native | Action

Creates an alert with a spacebar point (point that the camera will jump to when the player presses the spacebar) at the target point displaying the custom alert text.

# Arguments

  • string [ gamelink::Alert ] — Alert Type
  • int — Player
  • text — Text
  • string [ filepath ] — Icon
  • point — Point

Returns — void

native void UIAlertPoint(
	string alert,
	int player,
	text message,
	string icon,
	point p,
);
