# Remove Player From Player Group

Grammar — Remove player p from g|Player Group
Flags — Native | Action

Removes a player from the specified player group.

# Arguments

  • playergroup — Group
  • int — Player

Returns — void

native void PlayerGroupRemove(
	playergroup inGroup,
	int inPlayer,
