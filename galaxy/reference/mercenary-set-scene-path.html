# Set Mercenary Scene

Grammar — Set the Scene File Path of mercenaryId to scenePath
Flags — Native | Action

Sets the movie that plays when the specified mercenary is selected.

# Arguments

  • int [ preset::Mercenary ] — MercenaryId
  • string [ filepath ] — ScenePath

Returns — void

	int inMercenaryId,
	string inFilePath,
