# Set Dialog Transparency

Grammar — Set transparency of dialog to transparency%
Flags — Native | Action

Sets the transparency of the specified dialog.

# Arguments

  • int [ dialog ] — Dialog
  • fixed — Transparency

Returns — void

	int dialog,
	fixed inTransparency,
