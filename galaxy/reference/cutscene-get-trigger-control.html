# Dialog Item Cutscene

Grammar — Get Cutscene Id from dialogItem
Flags — Native | Function

Converts a Dialog Item of type Cutscene into a Cutscene that can be manipulated

# Arguments

Returns — int [ preset::Cutscene ]
