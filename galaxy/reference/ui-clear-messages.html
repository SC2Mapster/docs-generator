# Clear Text Messages

Grammar — Clear options for players
Flags — Native | Action

Clears all text messages currently displayed on the screen.

# Arguments

  • playergroup — Players
  • int [ preset::ClearMessageOptioins ] — Options

Returns — void

native void UIClearMessages(
	playergroup players,
	int messageArea,
);
