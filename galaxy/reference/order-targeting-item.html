# Order Targeting Item

Grammar — abilCmd targeting item
Flags — Native | Function

Returns an order targeting an item. This is primarily for use in “Issue Order” and “Issue Order To Unit Group” actions.

# Arguments

  • abilcmd — Ability Command
  • unit — Target Item

Returns — order

native order OrderTargetingItem(
	abilcmd inAbilCmd,
	unit inItem,
);
