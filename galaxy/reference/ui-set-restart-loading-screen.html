# Set Restart Loading Screen Tip

Grammar — Set the restart loading screen tip to help
Flags — Native | Action

Sets the loading screen tip to use when restarting the current map.

# Arguments

  • text — Help

Returns — void

native void UISetRestartLoadingScreen(text help);
