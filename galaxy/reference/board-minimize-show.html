# Show/Hide Leaderboard Minimize Button

Grammar — show|Show/Hide board|Leaderboard minimize button for players
Flags — Native | Action

Hides the minimize button on a leaderboard for a player group.

# Arguments

  • int [ preset::Board ] — Board
  • playergroup — Players
  • bool [ preset::ShowHideOption ] — Show

Returns — void

native void BoardMinimizeShow(
	int inBoard,
	playergroup inPlayers,
	bool inShow,
);
