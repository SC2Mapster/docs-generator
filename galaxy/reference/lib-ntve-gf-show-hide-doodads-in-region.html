# Show/Hide Doodads In Region

Grammar — showHide all doodadType doodads in the region target
Flags — Action

Show or hide all of the doodads of the specified type in a region.

# Arguments

  • bool [ preset::ShowHideOption ] — Show/Hide
  • region — Target
  • string [ gamelink::Actor ] — Doodad Type

Returns — void

void libNtve_gf_ShowHideDoodadsInRegion(
	bool lp_showHide,
	region lp_target,
	string lp_doodadType,
);
