# Debug Message

Grammar — Display msg as debug output using type, and ui|Do/Do Not display it in the game window
Flags — Native | Action

Outputs a debug message to a file, and optionally displays it to the screen as well. Use “Set Debug Message File” to configure the name of the output file.

# Arguments

  • int [ preset::DebugMessageType ] — Type
  • text — Text
  • bool [ preset::Do_Do_Not_Option ] — UI

Returns — void

native void TriggerDebugOutput(
	int type,
	text inText,
	bool includeGameUI,
);
