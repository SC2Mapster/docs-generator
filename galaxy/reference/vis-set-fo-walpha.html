# Set Fog of War Alpha

Flags — Native | Action

Override a player’s Fog of War transparency level. (0 = transparent, 100 = opaque)

# Arguments

  • int — Player
  • fixed — Alpha

Returns — void

native void VisSetFoWAlpha(int player, fixed alpha);
