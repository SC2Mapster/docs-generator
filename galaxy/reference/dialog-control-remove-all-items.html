# Remove All List Items

Grammar — Remove all items from list for players
Flags — Native | Action

Removes all of the list items from the specified dialog item.

# Arguments

  • int [ control ] — List
  • playergroup — Players

Returns — void

native void DialogControlRemoveAllItems(
	int control,
	playergroup players,
);
