# Trigger Is On

Grammar — t is on
Flags — Native | Function

Returns true if a trigger is turned on. Use the “Turn Trigger On/Off” action to turn a trigger on or off.

# Arguments

  • trigger — Trigger

Returns — bool

native bool TriggerIsEnabled(trigger t);
