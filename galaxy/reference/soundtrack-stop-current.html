# Stop Soundtrack (Current)

Grammar — Stop current category soundtrack for players (fade)
Flags — Native | Action

Stops a soundtrack for the specified player group. Queued soundtracks are not affected.

# Arguments

  • playergroup — Players
  • int [ preset::SoundtrackCategory ] — Category
  • bool [ preset::SoundFadeOption ] — Fade

Returns — void

native void SoundtrackStopCurrent(
	playergroup players,
	int category,
	bool fade,
);
