# Show/Hide Game UI

Grammar — showHide game UI for players
Flags — Action

Shows or hides the entire game UI. When shown, the UI will be set back to the visibility state of before it was hidden.

# Arguments

  • bool [ preset::ShowHideOption ] — Show/Hide
  • playergroup — Players

Returns — void

void libNtve_gf_HideGameUI(
	bool lp_showHide,
	playergroup lp_players,
);
