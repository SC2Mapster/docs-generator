# Stop Cutscene

Grammar — Stop inCutscene
Flags — Native | Action

Tells the specified Cutscene to stop playback. Stopping a cutscene destroys it.

# Arguments

  • int [ preset::Cutscene ] — Cutscene

Returns — void

native void CutsceneStop(int inCutscene);
