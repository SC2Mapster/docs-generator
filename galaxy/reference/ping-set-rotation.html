# Make Ping Face Angle

Grammar — Make ping face rotation degrees
Flags — Native | Action

Instantly rotates the ping to face a specific angle, given in degrees.

# Arguments

  • int [ ping ] — Ping
  • fixed — Rotation

Returns — void

native void PingSetRotation(int p, fixed rotation);
