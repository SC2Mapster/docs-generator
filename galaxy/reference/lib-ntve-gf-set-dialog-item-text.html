# Set Dialog Item Text

Grammar — Set dialogItem text to text for players
Flags — Action

Sets the text that appears within the dialog item.

# Arguments

  • int [ control ] — Dialog Item
  • text — Text
  • playergroup — Players

Returns — void

void libNtve_gf_SetDialogItemText(
	int lp_dialogItem,
	text lp_text,
	playergroup lp_players,
);
