# Show/Hide Help For Unit

Grammar — inShow|Show/Hide help for inUnit for player inPlayer
Flags — Native | Action

Determines whether or not a unit type will show up in the help panel.

# Arguments

  • int — Player
  • string [ gamelink::Unit ] — Unit
  • bool [ preset::ShowHideOption ] — Show

Returns — void

native void TechTreeUnitHelp(
	int player,
	string unitType,
	bool display,
);
