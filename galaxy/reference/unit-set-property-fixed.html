# Set Unit Property

Grammar — Set u prop to val
Flags — Native | Action

Sets a unit’s property.

# Arguments

  • unit — Unit
  • int [ preset::Unit_Property2 ] — Property
  • fixed — Value

Returns — void

	unit inUnit,
	int inProp,
	fixed inVal,
);
