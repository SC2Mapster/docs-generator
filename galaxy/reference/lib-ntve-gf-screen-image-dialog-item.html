# Screen Image Dialog Item

Grammar — Screen image screenImageID|ID dialog item
Flags — Function

Returns the dialog item that is the specified screen image. Use “Display Screen Image” to create screen images.

# Arguments

  • int — ScreenImageID

Returns — int [ control ]

int libNtve_gf_ScreenImageDialogItem(int lp_screenImageID);
