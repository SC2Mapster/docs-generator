# Model Transmission Source

Grammar — Model ModelLink
Flags — Native | Function

Applies a model as the transmission source for use in the transmission portrait.

# Arguments

  • string [ gamelink::Model ] — Model

Returns — transmissionsource

native transmissionsource TransmissionSourceFromModel(
	string modelLink,
);
