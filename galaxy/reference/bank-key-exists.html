# Bank Key Exists

Grammar — Bank bank has key in section section
Flags — Native | Function

Returns true if the chosen Bank has an entry at the specified Key of the specified Section.

# Arguments

  • bank — Bank
  • string — Section
  • string — Key

Returns — bool

native bool BankKeyExists(
	bank b,
	string section,
	string key,
);
