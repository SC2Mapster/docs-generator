# Remove All Players From Player Group

Grammar — Reset g|Player Group to have no players
Flags — Native | Action

Clears a player group so it no longer has any players in it.

# Arguments

  • playergroup — Group

Returns — void

native void PlayerGroupClear(playergroup inGroup);
