# Remove Charge Used For Player

Grammar — Clear charges used for charge inCharge for player inPlayer
Flags — Action

Clears the number of used charges for the specified player charge link.

# Arguments

  • int — Player
  • string — Charge

Returns — void

	string lp_inCharge,
