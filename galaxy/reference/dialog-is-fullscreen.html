# Dialog Is Fullscreen

Grammar — dialog is fullscreen
Flags — Native | Function

Returns true if the specified dialog is set to always be sized to the entire screen.

# Arguments

  • int [ dialog ] — Dialog

Returns — bool

native bool DialogIsFullscreen(int dialog);
