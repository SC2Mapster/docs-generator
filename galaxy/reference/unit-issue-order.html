# Issue Order

Grammar — Order u to ord (queue|Replace/After/Before Existing Orders)
Flags — Native | Action

Issues an order to a unit.

# Arguments

  • unit — Unit
  • order — Order
  • int [ preset::Unit_Order_Queue_Option ] — Queue

Returns — bool

native bool UnitIssueOrder(
	unit inUnit,
	order inOrder,
	int inQueueType,
);
