# Set Text Of Text Tag

Grammar — Set text of tag|Text Tag to text
Flags — Native | Action

Sets or changes the text of a text tag that has already been created.

# Arguments

  • int [ preset::TextTag ] — Tag
  • text — Text

Returns — void

native void TextTagSetText(int inTag, text inText);
