# Inventory Items Carried

Grammar — All items carried by unit
Flags — Native | Function

Returns a unit group containing all items carried by the specified unit.

# Arguments

  • unit — Unit

Returns — unitgroup

native unitgroup UnitInventoryGroup(unit inUnit);
