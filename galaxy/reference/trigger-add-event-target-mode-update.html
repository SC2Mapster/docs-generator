# Target Mode Updated

Grammar — Player player turns target mode state for abilityCommand
Flags — Native | Event

# Arguments

  • int — Player
  • abilcmd — Ability Command
  • int [ preset::TargetModeState ] — State

Returns — void

native void TriggerAddEventTargetModeUpdate(
	trigger t,
	int player,
	abilcmd inAbilCmd,
	int state,
);
