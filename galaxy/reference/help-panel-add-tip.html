# Create Tip

Grammar — Create a tip with title title, description description, alert text alertText and icon icon for players
Flags — Native | Action

Creates a tip that will display above the command card, and also adds the tip to the help panel.

# Arguments

  • playergroup — Players
  • text — Title
  • text — Description
  • text — Alert Text
  • string [ filepath ] — Icon

Returns — void

native void HelpPanelAddTip(
	playergroup players,
	text titleText,
	text descriptionText,
	text alertText,
	string iconPath,
);
