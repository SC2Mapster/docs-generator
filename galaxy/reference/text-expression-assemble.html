# Text Expression

Grammar — Text expression expressionId
Flags — Native | Function

Evalutes the given text expression with all token values in place. Token values may be set using “Set Text Expression Token”.

# Arguments

  • string — Expression Id

Returns — text

native text TextExpressionAssemble(string id);
