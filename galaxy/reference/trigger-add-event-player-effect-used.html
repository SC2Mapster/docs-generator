# Effect Used

Grammar — Player p uses Effect effect
Flags — Native | Event

This event fires when the specified effect is used by the specified player.

# Arguments

  • int — Player
  • string [ gamelink::Effect ] — Effect

Returns — void

native void TriggerAddEventPlayerEffectUsed(
	trigger t,
	int player,
	string inEffect,
);
