# Arctangent From Deltas

Grammar — Atan2(x, y)
Flags — Native | Function

Returns the arctangent of a set of X and Y deltas. The returned value is given in degrees.

# Arguments

  • fixed — Y
  • fixed — X

Returns — fixed

native fixed ATan2(fixed y, fixed x);
