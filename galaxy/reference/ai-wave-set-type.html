# Wave Set Type

Grammar — Sets the type type|WaveType and the target target|WaveTarget for wave wave
Flags — Native | Action

Sets the wave type and wave target for a wave.

# Arguments

  • wave — Wave
  • int — Type
  • wavetarget — Target

Returns — void

native void AIWaveSetType(
	wave w,
	int type,
	wavetarget target,
);
