# Show/Hide Objective

Grammar — show|Show/Hide objective for players|Player Group
Flags — Native | Action

Sets the state of an objective to either hidden or active.

# Arguments

  • int [ objective ] — Objective
  • playergroup — Players
  • bool [ preset::ShowHideOption ] — Show

Returns — void

native void ObjectiveShow(
	int inObjective,
	playergroup inPlayers,
	bool inShow,
);
