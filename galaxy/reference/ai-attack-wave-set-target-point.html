# Set Target Point For Attack Waves

Grammar — Set the target for player player attack waves to point
Flags — Native | Action

Sets a point to be the target of a player’s attack waves.

# Arguments

  • int — Attacker
  • point — Point

Returns — void

native void AIAttackWaveSetTargetPoint(
	int player,
	point p,
);
