# Player In Player Group

Grammar — p is in g|Player Group
Flags — Native | Function

Returns true if the specified player is in the specified player group.

# Arguments

  • playergroup — Group
  • int — Player

Returns — bool

native bool PlayerGroupHasPlayer(
	playergroup inGroup,
	int inPlayer,
);
