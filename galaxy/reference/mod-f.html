# Modulo (Real)

Grammar — x mod y
Flags — Native | Function

Uses real values. Returns the remainder after the dividend has been divided by the divisor.

# Arguments

  • fixed — Dividend
  • fixed — Divisor

Returns — fixed

native fixed ModF(fixed x, fixed m);
