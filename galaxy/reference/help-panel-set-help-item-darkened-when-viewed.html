# Set Help Item Darkened When Viewed

Grammar — Set the Darkened When Viewed Flag of helpItem to darkened
Flags — Native | Action

Flags the specified help item to darken when it’s marked as having been viewed by the player.

# Arguments

  • int [ preset::HelpItem ] — HelpItem
  • bool — Darkened

Returns — void

native void HelpPanelSetHelpItemDarkenedWhenViewed(
	int inHelpItemId,
	bool inDarkened,
);
