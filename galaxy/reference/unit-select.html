# Select/Deselect Unit

Grammar — select|Select/Deselect unit for player player
Flags — Native | Action

Selects or deselects a unit for a player.

# Arguments

  • unit — Unit
  • int — Player
  • bool [ preset::SelectDeselectOption ] — Select

Returns — void

native void UnitSelect(
	unit inUnit,
	int inPlayer,
	bool inSelect,
);
