# Set Title For Timer Window

Grammar — Set the title for window|Timer Window to title
Flags — Native | Action

Sets the title text for the specified timer window.

# Arguments

  • int [ preset::TimerWindow ] — Window
  • text — Title

Returns — void

native void TimerWindowSetTitle(
	int inWindow,
	text inTitle,
);
