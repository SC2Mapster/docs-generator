# Unit Order Is Acquired

Grammar — unit order at index index is Acquired
Flags — Native | Function

Returns true if the unit’s indexed order has been acquired rather than issued explicitly.

# Arguments

  • unit — Unit
  • int — Index

Returns — bool

native bool UnitOrderIsAcquired(unit inUnit, int inIndex);
