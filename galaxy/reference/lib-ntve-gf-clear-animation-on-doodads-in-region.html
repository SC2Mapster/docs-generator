# Clear Animation On Doodads In Region

Grammar — Clear the identifier animation for doodadType doodads in the region target
Flags — Action

Clears an animation with the chosen name (Identifier) on all doodads of the specified type in the specified region.

# Arguments

  • region — Target
  • string [ gamelink::Actor ] — Doodad Type
  • string — Identifier

Returns — void

void libNtve_gf_ClearAnimationOnDoodadsInRegion(
	region lp_target,
	string lp_doodadType,
	string lp_identifier,
);
