# Pause/Unpause Text Tag

Grammar — pause|Pause/Unpause tag|Text Tag
Flags — Native | Action

Pauses or unpauses any actions sent to the text tag.

# Arguments

  • int [ preset::TextTag ] — Tag
  • bool [ preset::PauseUnpauseOption ] — Pause

Returns — void

native void TextTagPause(int inTag, bool inPause);
