# Wave Add Unit Priority

Grammar — Add unit unit to wave wave with priority priority|Prio
Flags — Native | Action

Adds a unit to a wave with a priority.

# Arguments

  • wave — Wave
  • unit — Unit
  • int — Priority

Returns — void

native void AIWaveAddUnitPriority(
	wave w,
	unit u,
	int prio,
);
