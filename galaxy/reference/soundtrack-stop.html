# Stop Soundtrack (Current & Queued)

Grammar — Stop current & queued category soundtrack for players (fade)
Flags — Native | Action

Stops the current & queued soundtrack for the specified player group.

# Arguments

  • playergroup — Players
  • int [ preset::SoundtrackCategory ] — Category
  • bool [ preset::SoundFadeOption ] — Fade

Returns — void

native void SoundtrackStop(
	playergroup players,
	int category,
	bool fade,
);
