# Reload Bank

Grammar — Reload bank bank
Flags — Native | Action

Reloads the specified bank file, allowing you to access stored values later. If a bank has unsaved changes they will be lost when it reloads.

# Arguments

  • bank — Bank

Returns — void

native void BankReload(bank b);
