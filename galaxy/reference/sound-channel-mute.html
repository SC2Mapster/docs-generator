# Mute/Unmute Sound Channel

Grammar — mute|Mute/Unmute channel channel for players
Flags — Native | Action

Mutes or unmutes a sound channel for the specified player group.

# Arguments

  • playergroup — Players
  • int [ preset::SoundChannel ] — Channel
  • bool [ preset::MuteUnmuteOption ] — Mute

Returns — void

native void SoundChannelMute(
	playergroup players,
	int channel,
	bool mute,
);
