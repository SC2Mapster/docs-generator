# Remove Cutscene Filter

Grammar — Remove the filter inFilter from inCutscene.
Flags — Native | Action

Removes the filter for a cutscene. Any nodes with a non-empty filter property that does not match this value will be filtered out of your cutscene.

# Arguments

  • int [ preset::Cutscene ] — Cutscene
  • string — Filter

Returns — void

native void CutsceneRemoveFilter(
	int inCutscene,
	string inFilter,
);
