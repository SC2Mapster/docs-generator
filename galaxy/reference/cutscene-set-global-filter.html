# Set Global Cutscene Filter

Grammar — Set the global cutscene filter to inFilter.
Flags — Native | Action

Sets the filter for all cutscenes (removing any previous filters). Any nodes with a non-empty filter property that does not match this value will be filtered out of your cutscene. Newly created cutscenes will use this filter.

# Arguments

  • string — Filter

Returns — void

native void CutsceneSetGlobalFilter(string inFilter);
