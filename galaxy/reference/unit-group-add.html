# Add Unit To Unit Group

Grammar — Add u to g
Flags — Native | Action

Adds the specified unit to the specified unit group.

# Arguments

  • unitgroup — Unit Group
  • unit — Unit

Returns — void

native void UnitGroupAdd(unitgroup inGroup, unit inUnit);
