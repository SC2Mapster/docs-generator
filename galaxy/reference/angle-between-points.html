# Angle Between Points

Grammar — Angle from p1|Point to p2|Point
Flags — Native | Function

Returns the angle from point 1 to point 2 as a real value, in degrees.

# Arguments

  • point — Point 1
  • point — Point 2

Returns — fixed

native fixed AngleBetweenPoints(point p1, point p2);
