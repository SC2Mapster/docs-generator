# String Equal

Grammar — s1 is equal to s2 (sens|Sensitive/Insensitive to case)
Flags — Native | Function

Returns true if the two strings are the same.

# Arguments

  • string — String 1
  • string — String 2
  • bool [ preset::String_Case_Sensitivity ] — Sensitivity

Returns — bool

native bool StringEqual(
	string s1,
	string s2,
	bool caseSens,
);
