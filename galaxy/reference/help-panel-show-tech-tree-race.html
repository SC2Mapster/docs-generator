# Show/Hide Help Panel Tech Tree Race

Grammar — showHide the race tech tree button for players
Flags — Native | Action

Shows or hides a specific race’s tech tree on the help panel.

# Arguments

  • playergroup — Players
  • string [ gamelink::Race ] — Race
  • bool [ preset::ShowHideOption ] — Show/Hide

Returns — void

native void HelpPanelShowTechTreeRace(
	playergroup inPlayerGroup,
	string inRace,
	bool inShow,
);
