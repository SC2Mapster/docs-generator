# Show/Hide Ping

Grammar — visible|Show/Hide ping|Ping
Flags — Native | Action

Shows or hides a minimap ping.

# Arguments

  • int [ ping ] — Ping
  • bool [ preset::ShowHideOption ] — Visible

Returns — void

native void PingSetVisible(int p, bool isVisible);
