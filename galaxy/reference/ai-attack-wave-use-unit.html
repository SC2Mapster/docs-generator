# Add Unit To Attack Wave

Grammar — Add unit to the next attack wave for player player
Flags — Native | Action

Adds a unit to the next attack wave for a player.

# Arguments

  • int — Player
  • unit — Unit

Returns — void

native void AIAttackWaveUseUnit(int player, unit u);
