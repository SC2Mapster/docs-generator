# Wait For Transmission

Grammar — Wait until Transmission is Offset seconds from being finished
Flags — Native | Action

Waits for the specified transmission to finish.

# Arguments

  • int [ transmission ] — Transmission
  • fixed — Offset

Returns — void

native void TransmissionWait(int t, fixed offset);
