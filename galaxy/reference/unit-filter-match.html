# Unit Filter Match

Grammar — inUnit is inFilter for player inPlayer
Flags — Native | Function

Returns true if the specified unit matches the specified filter criteria from the perspective of the specified player.

# Arguments

  • unit — Unit
  • int — Player
  • unitfilter — Filter

Returns — bool

native bool UnitFilterMatch(
	unit inUnit,
	int inPlayer,
	unitfilter inFilter,
);
