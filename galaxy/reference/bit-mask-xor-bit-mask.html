# BitMaskXorBitMask

Flags — Native | Action

Xors the two BitMasks together placing the result in the Receiving BitMask (LHS = LHS ^ RHS)

# Arguments

  • bitmask — Receiving BitMask
  • bitmask — Xor With BitMask

Returns — void

native void BitMaskXorBitMask(bitmask lhs, bitmask rhs);
