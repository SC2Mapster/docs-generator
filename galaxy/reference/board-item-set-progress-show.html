# Show/Hide Leaderboard Item Progress Bar

Grammar — showHide progress bar for board|Leaderboard item at column column and row row
Flags — Native | Action

Shows or hides the display of a progress bar within a leaderboard item. Use the “Set Leaderboard Item Progress Value” action to adjust the progress bar’s displayed progress.

# Arguments

  • int [ preset::Board ] — Board
  • int — Column
  • int — Row
  • bool [ preset::ShowHideOption ] — Show/Hide

Returns — void

native void BoardItemSetProgressShow(
	int inBoard,
	int inCol,
	int inRow,
	bool inShow,
);
