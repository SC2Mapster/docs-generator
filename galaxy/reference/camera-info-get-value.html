# Camera Object Property

Grammar — c type
Flags — Native | Function

Returns the value of the specified Property for the chosen Camera Object.

# Arguments

  • camerainfo — Camera Object
  • int [ preset::CameraObjectPropertyType ] — Property

Returns — fixed

native fixed CameraInfoGetValue(camerainfo c, int type);
