# Show/Hide Timer Window

Grammar — show|Show/Hide window|Timer Window for players
Flags — Native | Action

Shows or hides the specified timer window.

# Arguments

  • int [ preset::TimerWindow ] — Window
  • playergroup — Players
  • bool [ preset::ShowHideOption ] — Show

Returns — void

native void TimerWindowShow(
	int inWindow,
	playergroup inPlayers,
	bool inShow,
);
