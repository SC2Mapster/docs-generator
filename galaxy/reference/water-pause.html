# Pause/Unpause Water Transition

Grammar — pauseUnpause water transition for waterType
Flags — Native | Action

The Set Water State trigger can be used to transition between two separate water states over a duration. This trigger will pause and resume that transition.

# Arguments

  • string [ gamelink::Water ] — Water Type
  • bool [ preset::PauseUnpauseOption ] — Pause/Unpause

Returns — void

native void WaterPause(string inWater, bool inPaused);
