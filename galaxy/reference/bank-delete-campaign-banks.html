# Delete Saved Campaign Banks

Grammar — Delete all saved campaign banks for campaign index for player player
Flags — Native | Action | Restricted

Deletes all the saved campaign banks specified in GameUIData for the Campaign specified by the campaign index.

# Arguments

  • int — Player
  • int — Index

Returns — void

native void BankDeleteCampaignBanks(int player, int index);
