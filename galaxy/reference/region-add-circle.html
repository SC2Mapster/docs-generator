# Add Circle To Region

Grammar — Add a state|Positive/Negative circle centered at c with radius rad to r
Flags — Native | Action

Adds a circle to a region.

# Arguments

  • region — Region
  • bool [ preset::Region_State ] — State
  • point — Center
  • fixed — Radius

Returns — void

native void RegionAddCircle(
	region r,
	bool positive,
	point center,
	fixed radius,
);
