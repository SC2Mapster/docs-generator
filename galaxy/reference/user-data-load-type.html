# Load User Data Type

Grammar — Load user data for userType from bank bank section section
Flags — Native | Action

Loads all modified values for all instances of the given type.

# Arguments

  • string [ gamelink::User ] — User Type
  • bank — Bank
  • string — Section

Returns — void

native void UserDataLoadType(
	string inType,
	bank inBank,
	string inSection,
);
