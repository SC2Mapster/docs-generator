# Stack Count Of Behavior On Unit

Grammar — Stack count of behavior on unit
Flags — Native | Function

Returns the stack count of a behavior on a unit.

# Arguments

  • unit — Unit
  • string [ gamelink::Behavior ] — Behavior

Returns — int
