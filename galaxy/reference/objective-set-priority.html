# Set Objective Priority

Grammar — Set Objective objective Priority to priority
Flags — Native | Action

Determines which order objectives will display in. Larger priorities will appear higher in the list. The default priority for an objective is -1.

# Arguments

  • int [ objective ] — Objective
  • int — Priority

Returns — void

native void ObjectiveSetPriority(
	int inObjective,
	int inPriority,
);
