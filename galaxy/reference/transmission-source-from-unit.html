# Unit Transmission Source

Grammar — Unit with Flash|Flash Option (overridePortrait|Do/Do Not override portrait) playing anim|Animation
Flags — Native | Function

Applies a unit as the transmission source for use in the transmission portrait.

# Arguments

  • unit — Unit
  • bool [ preset::TransmissionFlashOption ] — Flash
  • bool [ preset::Do_Do_Not_Option ] — Override Portrait
  • string [ modelanim ] — Anim

Returns — transmissionsource

native transmissionsource TransmissionSourceFromUnit(
	unit u,
	bool flash,
	bool overridePortrait,
	string anim,
);
