# Elapsed Time Of Timer

Grammar — Elapsed time of t
Flags — Native | Function

Returns the amount of time that has elapsed for the specified timer.

# Arguments

  • timer — Timer

Returns — fixed

native fixed TimerGetElapsed(timer t);
