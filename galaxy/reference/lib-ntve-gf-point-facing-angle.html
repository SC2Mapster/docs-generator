# Point Facing Angle

Grammar — point facing angle degrees
Flags — Function

Returns a point with the position of the specified point, and the facing of the specified angle.

# Arguments

  • point — Point
  • fixed — Angle

Returns — point

point libNtve_gf_PointFacingAngle(
	point lp_point,
	fixed lp_angle,
);
