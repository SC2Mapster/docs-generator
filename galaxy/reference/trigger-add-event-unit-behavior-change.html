# Unit Behavior Changes

Grammar — unit has behavior change type
Flags — Native | Event

# Arguments

  • unitref [ unit ] — Unit
  • string [ gamelink::Behavior ] — Behavior
  • int [ preset::BehaviorChange ] — Type

Returns — void

	trigger t,
	unitref u,
	string behavior,
	int inChangeType,
);
