# Revive Order Targeting Point

Grammar — abilCmd targeting p reviving unit reviveUnit
Flags — Function

Returns an order targeting a point and will revive a specific unit. This is primarily for use in “Issue Order To Unit Group” actions.

# Arguments

  • abilcmd — Ability Command
  • point — Target Point
  • unit — Revive Unit

Returns — order

order libNtve_gf_ReviveOrderTargetingPoint(
	abilcmd lp_abilCmd,
	point lp_p,
	unit lp_reviveUnit,
);
