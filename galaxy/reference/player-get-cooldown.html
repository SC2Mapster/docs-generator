# Player Cooldown

Grammar — Cooldown of cooldown inCooldown for player inPlayer
Flags — Native | Function

Returns the cooldown time for the specified player cooldown link.

# Arguments

  • int — Player
  • string — Cooldown

Returns — fixed

native fixed PlayerGetCooldown(
	int inPlayer,
	string inCooldown,
);
