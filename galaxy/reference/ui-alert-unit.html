# Create Alert At Unit

Grammar — Create a alertType|Alert Type alert at unit for player player with the text: text (Custom icon: icon)
Flags — Native | Action

Creates an alert with a spacebar point (point that the camera will jump to when the player presses the spacebar) at the target unit displaying the custom alert text.

# Arguments

  • string [ gamelink::Alert ] — Alert Type
  • int — Player
  • text — Text
  • string [ filepath ] — Icon
  • unit — Unit

Returns — void

native void UIAlertUnit(
	string alert,
	int player,
	text message,
	string icon,
	unit u,
);
