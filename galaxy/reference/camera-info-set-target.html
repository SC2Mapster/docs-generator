# Set Camera Object Target

Grammar — Set c|Camera Object target to p|Point
Flags — Native | Action

Sets a camera object to a selected point.

# Arguments

  • camerainfo — CameraObject
  • point — Target

Returns — void

native void CameraInfoSetTarget(camerainfo c, point p);
