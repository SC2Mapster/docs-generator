# Hookup Standard Dialog Item

Grammar — Hooks up an existing type in the standard UI called name
Flags — Native | Action

Hooks up an already existing dialog item in the default UI of the specified type. This allows dynamic modification of any frame within the Built-In UI. Frames that were created from locked Descs cannot be hooked up.

# Arguments

  • int [ preset::ControlType ] — Type
  • string — Name

Returns — int [ control ]

native int DialogControlHookupStandard(
	int type,
	string inTemplate,
);
