# Game Is Transitioned

Grammar — Game is transitioned
Flags — Native | Function

Returns true if the game is transistioned. A game is transitioned if it has started as a result of a previous map (IE: Was not simply loaded from a lobby). Only supported in single player offline games.

Returns — bool

native bool GameIsTransitionMap();
