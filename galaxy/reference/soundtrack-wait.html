# Wait For Soundtrack

Grammar — Wait for soundtrack playback to end
Flags — Native | Action

Waits for soundtrack playback to end.

# Arguments

  • string [ gamelink::Soundtrack ] — Soundtrack

Returns — void

native void SoundtrackWait(string soundtrack);
