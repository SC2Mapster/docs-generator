# Change Unit Height

Grammar — Change u height to height over dur|Time seconds
Flags — Native | Action

Changes the specified unit’s height.

# Arguments

  • unit — Unit
  • fixed — Height
  • fixed — Duration

Returns — void

native void UnitSetHeight(
	unit inUnit,
	fixed inHeight,
	fixed inDuration,
);
