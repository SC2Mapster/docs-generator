# Attach Ping To Unit

Grammar — Attach ping to unit
Flags — Native | Action

Sets the ping location on the mini map to the position of the specified unit.

# Arguments

  • int [ ping ] — Ping
  • unit — Unit

Returns — void

native void PingSetUnit(int p, unit u);
