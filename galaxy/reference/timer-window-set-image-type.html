# Set Timer Window Image Type

Grammar — Set window|Timer Window image image type to imageType
Flags — Native | Action

Sets the image type within a timer window. The image files may be customized in Assets.txt, under UI/TimerWindow.

# Arguments

  • int [ preset::TimerWindow ] — Window
  • int [ preset::TimerWindowImage ] — Image
  • int [ preset::ImageType ] — Image Type

Returns — void

native void TimerWindowSetImageType(
	int inWindow,
	int inImage,
	int inType,
);
