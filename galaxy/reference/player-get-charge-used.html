# Player Charge Used

Grammar — Charges used of charge inCharge for player inPlayer
Flags — Native | Function

Returns the number of charges used for the specified player charge link.

# Arguments

  • int — Player
  • string — Charge

Returns — fixed

native fixed PlayerGetChargeUsed(
	int inPlayer,
	string inCharge,
);
