# Set Ping Tooltip

Grammar — Set ping tooltip to tooltip
Flags — Native | Action

Sets a ping’s tooltip, which is displayed when a user points at the ping with the mouse.

# Arguments

  • int [ ping ] — Ping
  • text — Tooltip

Returns — void

native void PingSetTooltip(int p, text tooltip);
