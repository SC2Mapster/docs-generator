# Set Dialog Size

Grammar — Set size of dialog to (width, height)
Flags — Native | Action

Sets the width and height of a dialog.

# Arguments

  • int [ dialog ] — Dialog
  • int — Width
  • int — Height

Returns — void

native void DialogSetSize(
	int dialog,
	int width,
	int height,
);
