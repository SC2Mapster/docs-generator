# Set Objective State

Grammar — Mark objective as state
Flags — Native | Action

Sets the state of an objective.

# Arguments

  • int [ objective ] — Objective
  • int [ preset::ObjectiveState ] — State

Returns — void

native void ObjectiveSetState(
	int inObjective,
	int inState,
);
