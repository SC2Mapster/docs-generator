# Show/Hide World

Grammar — isVisible|Show/Hide World for players
Flags — Native | Action

Shows or hides the 3D game world for the players specified.

# Arguments

  • playergroup — Players
  • bool [ preset::ShowHideOption ] — Show

Returns — void

native void UISetWorldVisible(
	playergroup players,
	bool isVisible,
);
