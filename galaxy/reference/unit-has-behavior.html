# Unit Has Behavior (Enabled)

Grammar — unit has behavior
Flags — Native | Function

Returns true if the stack count for this behavior is > 0

# Arguments

  • unit — Unit
  • string [ gamelink::Behavior ] — Behavior

Returns — bool

native bool UnitHasBehavior(
	unit inUnit,
	string inBehavior,
);
