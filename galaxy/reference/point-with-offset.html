# Point With Offset

Grammar — p offset by (x, y)
Flags — Native | Function

Returns a point offset by given horizontal and vertical distances.

# Arguments

  • point — Point
  • fixed — X
  • fixed — Y

Returns — point

native point PointWithOffset(point p, fixed x, fixed y);
