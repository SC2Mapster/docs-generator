# BitMaskSetIndex

Flags — Native | Action

Sets a flag on the bit mask either on or off depending on the boolean Set On. Returns true if the setting worked.

# Arguments

  • bitmask — Bit Mask
  • int — Index to Set
  • bool — Set On

Returns — void

native void BitMaskSetIndex(
	bitmask mask,
	int index,
	bool on,
);
