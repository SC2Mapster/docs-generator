# Load User Data Value Into Variable

Grammar — Set var = User Data Type userDataType: User Data (userType, instance, field, index)
Flags — Operator | Action

Returns a value from user data for the given type, instance, and field. User data is defined in the Data module.

# Arguments

  • string [ gamelink::User ] — User Type
  • string [ userinstance ] — Instance
  • string [ userfield ] — Field
  • int — Index
  • anyvariable — Variable
  • int [ preset::UserDataTypeFunctionRefGet ] — User Data Type

Returns — abilcmd

#PARAM(var) = #PARAM(userDataType)(#PARAM(userType),#PARAM(instance),#PARAM(field), #PARAM(index));
