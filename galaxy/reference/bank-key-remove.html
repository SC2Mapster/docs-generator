# Remove Bank Key

Grammar — Remove key from section section in bank bank
Flags — Native | Action

Removes a specific bank Key from a section in the chosen bank.

# Arguments

  • bank — Bank
  • string — Section
  • string — Key

Returns — void

native void BankKeyRemove(
	bank b,
	string section,
	string key,
);
