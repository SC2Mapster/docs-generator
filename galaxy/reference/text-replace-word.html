# Text With Search Replacement

Grammar — text with up to count findText replaced by replaceText (sensitivity|Sensitive/Insensitive to case)
Flags — Native | Function

Replaces one or more occcurrences of a substring inside text.

# Arguments

  • text — Text
  • text — Find Text
  • text — Replace Text
  • int — Count
  • bool [ preset::String_Case_Sensitivity ] — Sensitivity

Returns — text

native text TextReplaceWord(
	text t,
	text word,
	text replace,
	int maxCount,
	bool caseSens,
);
