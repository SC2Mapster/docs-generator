# Set Dialog Item Size

Grammar — Set dialogItem size to (width, height) for players
Flags — Native | Action

Sets the width and height of a dialog item.

# Arguments

  • int [ control ] — Dialog Item
  • playergroup — Players
  • int — Width
  • int — Height

Returns — void

native void DialogControlSetSize(
	int control,
	playergroup players,
	int width,
	int height,
);
