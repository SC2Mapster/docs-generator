# AI Pathing Cost Map

Grammar — Pathing cost from point from to point to
Flags — Native | Function

Returns the map pathing cost between two points, ignores all units.

# Arguments

  • point — From
  • point — To

Returns — int

native int AIPathingCostMap(point from, point to);
