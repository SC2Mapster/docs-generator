# Unit Tracked Group

Grammar — Tracked Unit Group of inUnit for behavior inConjoinedBehavior
Flags — Native | Function

Returns the tracked unit group of a unit given a unit tracker behavior

# Arguments

  • unit — Tracker Unit
  • string [ gamelink::Behavior ] — Unit Tracker Behavior

Returns — unitgroup

native unitgroup UnitGetTrackedUnitGroup(
	unit inUnit,
	string inConjoinedBehavior,
);
