# Triggering Cargo Unit

Grammar — Triggering cargo unit
Flags — Native | Function

Returns the triggering cargo unit. Used in response to a “Unit Loads/Unloads Cargo” event.

Returns — unit

native unit EventUnitCargo();

# Supported triggers

  • Unit Loads/Unloads Cargo — TriggerAddEventUnitCargo
