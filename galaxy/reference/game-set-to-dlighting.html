# Set Time of Day Lighting

Grammar — Change the light used for time of day lighting to light
Flags — Native | Action

Sets the lighting used by the time of day.

# Arguments

  • string [ gamelink::Light ] — Light

Returns — void

native void GameSetToDLighting(string inId);
