# Set Screen Mode

Grammar — Set screen mode to mode for players over duration seconds
Flags — Native | Action

Changes the screen mode for the player group over the duration specified.

# Arguments

  • playergroup — Players
  • int [ preset::ScreenMode ] — Mode
  • fixed — Duration

Returns — void

native void UISetMode(
	playergroup players,
	int mode,
	fixed duration,
);
