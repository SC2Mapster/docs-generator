# Mid-Point

Grammar — Mid-Point of sourcePoint and targetPoint
Flags — Native | Function

Returns the point half-way between the two given points.

# Arguments

  • point — Source Point
  • point — Target Point

Returns — point

native point MidPoint(point source, point dest);
