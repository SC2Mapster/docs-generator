# Show/Hide Dialog Background

Grammar — visible|Show/Hide the background image of dialog
Flags — Native | Action

Shows or hides a dialog’s background. Dialog items will remain visible.

# Arguments

  • int [ dialog ] — Dialog
  • bool [ preset::ShowHideOption ] — Visible

Returns — void

native void DialogSetImageVisible(
	int dialog,
	bool isVisible,
);
