# Movie Transmission Source

Grammar — Movie movie with subtitles Subtitles
Flags — Native | Function

Applies a movie as the transmission source for use in the transmission portrait.

# Arguments

  • string [ filepath ] — Movie
  • bool [ preset::OnOffOption ] — Subtitles

Returns — transmissionsource

native transmissionsource TransmissionSourceFromMovie(
	string assetLink,
	bool subtitles,
);
