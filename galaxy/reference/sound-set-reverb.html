# Set Sound Reverb (Deprecated)

Grammar — Set sound reverb to reverb over duration seconds (Ambient: ambient, Global: global)
Flags — Native | Action | Hidden

Sets sound reverb.

# Arguments

  • string [ gamelink::Reverb ] — Reverb

  • bool — Ambient
  • bool — Global

Returns — void

	string inReverbLink,
	fixed inDuration,
	bool inAmbient,
	bool inGlobal,
