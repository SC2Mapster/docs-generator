# Set Timer Window Color

Grammar — Set window|Timer Window component color to color with transparency% transparency
Flags — Native | Action

Sets the color and transparency for the specified timer window.

# Arguments

  • int [ preset::TimerWindow ] — Window
  • int [ preset::TimerWindowComponent ] — Component
  • color — Color
  • fixed — Transparency

Returns — void

native void TimerWindowSetColor(
	int inWindow,
	int inType,
	color inColor,
	fixed transparency,
);
