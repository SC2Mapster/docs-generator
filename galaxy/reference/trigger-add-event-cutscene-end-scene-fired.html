# Cutscene EndScene Fired

Grammar — Cutscene cutscene fired End Scene event
Flags — Native | Event

When the cutscene ends this event is triggered, this event is fired so that triggers may respond to it.

# Arguments

  • int [ preset::Cutscene ] — Cutscene

Returns — void

native void TriggerAddEventCutsceneEndSceneFired(
	trigger t,
	int inCutscene,
);
