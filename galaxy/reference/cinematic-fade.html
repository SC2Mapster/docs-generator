# Fade In/Out

Grammar — fadeIn|Fade In/Fade Out over duration seconds using color color and transparency% transparency (style style, waitUntilDone|Wait/Don’t Wait until it finishes)
Flags — Native | Action

Fades the screen in or out over a specified amount of time. If you want to fade in from a Cutscene, use Cutscene fade.

# Arguments

  • bool [ preset::FadeOption ] — Fade In
  • fixed — Duration
  • int [ preset::FadeStyle ] — Style
  • color — Color
  • fixed — Transparency
  • bool [ preset::Trigger_Wait_Option ] — Wait Until Done

Returns — void

native void CinematicFade(
	bool fadeIn,
	fixed duration,
	int style,
	color inColor,
	fixed transparency,
	bool waitUntilDone,
);
