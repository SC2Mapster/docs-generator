# Pause Soundtrack

Grammar — pause category soundtrack for players (fade|Fade Option)
Flags — Native | Action

Pauses a soundtrack for the specified player group.

# Arguments

  • playergroup — Players
  • int [ preset::SoundtrackCategory ] — Category
  • bool [ preset::PauseUnpauseOption ] — Pause
  • bool [ preset::SoundFadeOption ] — Fade

Returns — void

native void SoundtrackPause(
	playergroup players,
	int category,
	bool pause,
	bool fade,
);
