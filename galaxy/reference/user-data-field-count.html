# Number Of User Data Fields

Grammar — Number of user data fields for userType
Flags — Native | Function

Returns the number of fields defined within the given user type. User types are defined within the Data module.

# Arguments

  • string [ gamelink::User ] — User Type

Returns — int

native int UserDataFieldCount(string inType);
