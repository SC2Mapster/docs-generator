# Set Debug Message Filter

Grammar — enabled|Enable the filter filter for debug message type type
Flags — Native | Action

Enable or disable various filters for debug messages. By default all filters are off and must be enabled if you want filtering to occur.

# Arguments

  • int [ preset::DebugMessageType ] — Type
  • int [ preset::DebugMessageFilter ] — Filter
  • bool [ preset::Enable_Disable_Option ] — Enabled

Returns — void

native void TriggerDebugSetTypeFilter(
	int type,
	int filter,
	bool enabled,
);
