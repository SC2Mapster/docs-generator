# Pulse Screen Image

Grammar — Pulse screen image screenImageID between transparency1 and transparency2 every period seconds
Flags — Action

Causes a screen image to change between two transperency values at the specified interval.

# Arguments

  • int — ScreenImageID
  • fixed — Period
  • fixed — Transparency1
  • fixed — Transparency2

Returns — void

void libNtve_gf_PulseScreenImage(
	int lp_screenImageID,
	fixed lp_period,
	fixed lp_transparency1,
	fixed lp_transparency2,
);
