# Set Dialog Title

Grammar — Set title of dialog to title
Flags — Native | Action

Sets the text that is used for the title of the dialog.

# Arguments

  • int [ dialog ] — Dialog
  • text — Title

Returns — void

native void DialogSetTitle(int dialog, text title);
