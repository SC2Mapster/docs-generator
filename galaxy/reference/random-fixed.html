# Random Real

Grammar — Random real between min and max
Flags — Native | Function

Returns a random real between the specified values (inclusive).

# Arguments

  • fixed — Min
  • fixed — Max

Returns — fixed

native fixed RandomFixed(fixed min, fixed max);
