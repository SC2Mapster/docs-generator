# Set Fog Color Over Time

Grammar — Set fog’s color to color over duration seconds
Flags — Native | Action

Sets the color of environmental fog.

# Arguments

  • color — Color
  • fixed — Duration

Returns — void

native void FogSetColorOverTime(color c, fixed timeRamp);
