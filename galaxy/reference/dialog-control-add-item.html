# Add List Item

Grammar — Add list item text to list for players
Flags — Native | Action

Adds a list item to a dialog item with a list, such as a pulldown or a list box.

# Arguments

  • int [ control ] — List
  • playergroup — Players
  • text — Text

Returns — void

native void DialogControlAddItem(
	int control,
	playergroup players,
	text value,
);
