# Send Actor Message To Game Region

Grammar — Send actor message message to all actors contained by game region region
Flags — Action

Sends an actor message to all the actors contained within the specified game region. An actor message will perform an action and/or trigger an event on the actor it is sent to.

# Arguments

  • region — Region
  • string [ actormsg ] — Message

Returns — void

void libNtve_gf_SendActorMessageToGameRegion(
	region lp_region,
	string lp_message,
);
