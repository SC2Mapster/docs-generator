# Pause/Unpause Time Of Day

Grammar — pause|Pause/Unpause time of day cycle
Flags — Native | Action

Pauses or unpauses the time of day cycle.

# Arguments

  • bool [ preset::PauseUnpauseOption ] — Pause

Returns — void

native void GameTimeOfDayPause(bool inPause);
