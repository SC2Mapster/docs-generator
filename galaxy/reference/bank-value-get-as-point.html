# Load Point Value

Grammar — Load key of section section from bank bank as point value
Flags — Native | Function

Returns a Point value contained in the chosen Bank at the specified Key of the specified Section.

# Arguments

  • bank — Bank
  • string — Section
  • string — Key

Returns — point
