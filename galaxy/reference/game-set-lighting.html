# Set Lighting

Grammar — Set lighting to light, blending over blendTime seconds
Flags — Native | Action

Changes the lighting on the map.

# Arguments

  • string [ gamelink::Light ] — Light
  • fixed — Blend Time

Returns — void

native void GameSetLighting(
	string inId,
	fixed inBlendTime,
);
