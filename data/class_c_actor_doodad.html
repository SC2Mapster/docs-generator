Boosted Cliff Level.

The footprint for a doodad can flag cells as Boost Cliff. Any cells with the Boost Cliff flag will adjust the cliff level used for vision to the cliff level specified here. Typically used to adjust the cliff level of bridge doodads.

◆ BoostedHeight

std::vector<CFixed> CActorDoodad::BoostedHeight

Boosted Height.

The footprint for a doodad can flag cells as Boost Height. Any cells with the Boost Height flag will adjust the synchronous height map to the specified height. Typically used on large tree doodads such that flying units will fly over the trees rather than through them.

minValue: 0
