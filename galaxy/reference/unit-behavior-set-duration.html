# Set Unit Behavior Duration

Grammar — Set behavior duration to duration on unit unit
Flags — Native | Action

Sets a behavior duration on a unit.

# Arguments

  • unit — Unit
  • string [ gamelink::Behavior ] — Behavior
  • fixed — Duration

Returns — void

native void UnitBehaviorSetDuration(
	unit inUnit,
	string inBehavior,
	fixed inDuration,
);
