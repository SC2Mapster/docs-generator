# Remove Unit From Unit Group

Grammar — Remove u from g
Flags — Native | Action

Removes the specified unit from the specified unit group.

# Arguments

  • unitgroup — Unit Group
  • unit — Unit

Returns — void

native void UnitGroupRemove(
	unitgroup inGroup,
	unit inUnit,
);
