# Cheat Used

Grammar — Player p uses type cheat code.
Flags — Native | Event

This event fires when the specified player uses the specified cheat. Use “Cheat Type Used” to get the cheat that was used.

# Arguments

  • int — Player
  • int [ preset::GameCheatType ] — Type

Returns — void

native void TriggerAddEventCheatUsed(
	trigger t,
	int player,
	int inCheat,
);

# Event parameters

  • Cheat Type Used — EventCheatUsed
  • Player Ping Unit — EventPingUnit
  • Player Ping Unit Type — EventPingUnitType
  • Player Ping Unit ControlPlayer — EventPingUnitControlPlayer
  • Player Ping Unit UpkeepPlayer — EventPingUnitUpkeepPlayer
  • Player Ping Unit Position — EventPingUnitPosition
  • Player Ping Location — EventPingPoint
  • Player Ping Is From Minimap — EventPingedMinimap
  • Player Ping Option — EventPingOption
