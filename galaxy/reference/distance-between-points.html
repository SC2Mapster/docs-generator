# Distance Between Points

Grammar — Distance between p1|Point and p2|Point
Flags — Native | Function

Returns the distance between two points as a real value.

# Arguments

  • point — Point 1
  • point — Point 2

Returns — fixed

native fixed DistanceBetweenPoints(point p1, point p2);
