# Convert String Case

Grammar — String(s) as case|Lower/Upper Case
Flags — Native | Function

Converts a string to upper case or lower case.

# Arguments

  • string — String
  • bool [ preset::String_Case_Option ] — Case

Returns — string

native string StringCase(string s, bool upper);
