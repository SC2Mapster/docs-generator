# Convert Real To Text

Grammar — Text(val|Real) with prec decimal places
Flags — Native | Function

Converts a real value to a text value.

# Arguments

  • fixed — Value
  • int — Precision

Returns — text

native text FixedToText(fixed x, int precision);
