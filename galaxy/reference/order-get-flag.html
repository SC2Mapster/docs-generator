# Flag Is Enabled For Order

Grammar — flag is enabled for order
Flags — Native | Function

Returns true if the specified flag is enabled.

# Arguments

  • order — Order
  • int [ preset::OrderFlag ] — Flag

Returns — bool

native bool OrderGetFlag(order inOrder, int inFlag);
