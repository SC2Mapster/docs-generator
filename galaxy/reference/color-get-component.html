# Color Component

Grammar — component percent of c
Flags — Native | Function

Returns the percentage value of a color component in the specified color.

# Arguments

  • color — Color
  • int [ preset::colorComponent ] — Component

Returns — fixed

native fixed ColorGetComponent(color c, int component);
