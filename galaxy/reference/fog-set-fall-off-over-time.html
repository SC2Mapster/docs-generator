# Set Fog Falloff Over Time

Grammar — Set fog’s falloff to falloff over duration seconds
Flags — Native | Action

Sets the falloff for environmental fog. Falloff determines how far from the fog start height the environmental fog will extend.

# Arguments

  • fixed — Falloff
  • fixed — Duration

Returns — void

native void FogSetFallOffOverTime(fixed f, fixed timeRamp);
