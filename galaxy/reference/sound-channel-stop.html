# Stop Sound Channel

Grammar — Stop channel channel for players
Flags — Native | Action

Stops all sounds in a sound channel for the specified players.

# Arguments

  • playergroup — Players
  • int [ preset::SoundChannel ] — Channel

Returns — void

native void SoundChannelStop(
	playergroup players,
	int channel,
);
