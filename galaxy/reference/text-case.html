# Convert Text Case

Grammar — Text(t) as case|Lower/Upper Case
Flags — Native | Function

Converts text to upper case or lower case.

# Arguments

  • text — Text
  • bool [ preset::String_Case_Option ] — Case

Returns — text

native text TextCase(text t, bool upper);
