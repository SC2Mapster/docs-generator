# Unit Is Selected

Grammar — unit is selected by player player
Flags — Native | Function

Returns true if the specified unit is selected by the specified player.

# Arguments

  • unit — Unit
  • int — Player

Returns — bool

native bool UnitIsSelected(unit inUnit, int inPlayer);
