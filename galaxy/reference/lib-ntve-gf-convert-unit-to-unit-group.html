# Convert Unit To Unit Group

Grammar — Unit group(unit)
Flags — Function

Takes one unit and creates a unit group out of it.

# Arguments

  • unit — Unit

Returns — unitgroup

unitgroup libNtve_gf_ConvertUnitToUnitGroup(unit lp_unit);
