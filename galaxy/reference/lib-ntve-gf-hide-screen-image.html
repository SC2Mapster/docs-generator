# Show/Hide Screen Image

Grammar — showHideOption|Show/Hide screen image screenImageID
Flags — Action

Shows or hides a screen image. Use “Display Screen Image” to create screen images.

# Arguments

  • bool [ preset::ShowHideOption ] — ShowHideOption
  • int — ScreenImageID

Returns — void

void libNtve_gf_HideScreenImage(
	bool lp_showHideOption,
	int lp_screenImageID,
);
