# Default Economy

Flags — Native | Action

Sets stock for a player’s economic buildings and units based on the specified parameters.

# Arguments

  • int — Player
  • string [ gamelink::Unit ] — TownHall
  • string [ gamelink::Unit ] — Refinery
  • string — Food
  • string — Peon
  • int — Cap
  • bool — peonMode

Returns — void

native void AIDefaultEconomy(
	int player,
	string hall,
	string gas,
	string food,
	string peon,
	int cap,
	bool peonMode,
);
