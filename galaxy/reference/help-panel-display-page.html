# Display Help Page

Grammar — Display the help panel page for players
Flags — Native | Action

Shows the help panel set to the specified page for the specified players.

# Arguments

  • playergroup — Players
  • int [ preset::HelpPage ] — Page

Returns — void

native void HelpPanelDisplayPage(
	playergroup players,
	int inPage,
);
