# Player Leaves Game

Grammar — Player p leaves the game with result|Leave Game Option
Flags — Native | Event

This event fires when a player leaves the game with the specified victory type. Use “Triggering Player” to get the player.

# Arguments

  • int — Player
  • int [ preset::LeaveGameOption ] — Result

Returns — void

native void TriggerAddEventPlayerLeft(
	trigger inTrigger,
	int player,
	int inResult,
);

# Event parameters

  • Triggering Game User — EventGameUser
  • Triggering Player — EventPlayer
