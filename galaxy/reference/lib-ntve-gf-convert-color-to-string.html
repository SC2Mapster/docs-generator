# Convert Color To String

Grammar — Convert color color to string
Flags — Function

Converts the RGB values of the specified Color to a string value with the format “,,”.

# Arguments

  • color — Color

Returns — string

string libNtve_gf_ConvertColorToString(color lp_color);
