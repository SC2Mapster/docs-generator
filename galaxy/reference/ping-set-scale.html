# Set Ping Scale

Grammar — Set scale of ping to scale
Flags — Native | Action

Sets the size of the ping.

# Arguments

  • int [ ping ] — Ping
  • fixed — Scale

Returns — void

native void PingSetScale(int p, fixed scale);
