# Unit Group Is Dead

Grammar — All units in units|Unit Group are dead
Flags — Function

Returns true if all units in the unit group are dead. An empty unit group is considered to be dead.

# Arguments

  • unitgroup — Units

Returns — bool

bool libNtve_gf_UnitGroupIsDead(unitgroup lp_units);
