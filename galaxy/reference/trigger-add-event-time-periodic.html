# Periodic Event

Grammar — Every dur|Time seconds of timeType|Real/Game/AI Time
Flags — Native | Event

This event fires periodically whenever the time specified elapses.

# Arguments

  • fixed — Duration
  • int [ preset::TimeType ] — Time Type

Returns — void

native void TriggerAddEventTimePeriodic(
	trigger inTrigger,
	fixed inTime,
	int inTimeType,
);
