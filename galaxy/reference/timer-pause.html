# Pause/Unpause Timer

Grammar — pause|Pause/Resume t
Flags — Native | Action

Pauses or resumes a timer.

# Arguments

  • timer — Timer
  • bool [ preset::PauseUnpauseOption ] — Pause

Returns — void

native void TimerPause(timer t, bool pause);
