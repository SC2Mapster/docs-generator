# Unit Has Behavior

Grammar — unit has behavior
Flags — Native | Function

Returns true if the specified unit has the specified behavior.

# Arguments

  • unit — Unit
  • string [ gamelink::Behavior ] — Behavior

Returns — bool

native bool UnitHasBehavior2(
	unit inUnit,
	string inBehavior,
);
