# Value From Data Table (Unit) - Instance

Grammar — name from the data table instance instance
Flags — Native | Function

Returns a unit value from a data table instace. Data tables allow you to store and recall values using a string identifier.

# Arguments

  • int [ datatable ] — Instance
  • string — Name

Returns — unit

native unit DataTableInstanceGetUnit(
	int instance,
	string name,
);
