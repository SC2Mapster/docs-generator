# Attach Region To Unit

Grammar — Attach region to unit with offset offset
Flags — Native | Action

Attaches a region to a unit.

# Arguments

  • region — Region
  • unit — Unit
  • point — Offset

Returns — void

native void RegionAttachToUnit(
	region r,
	unit u,
	point offset,
);
