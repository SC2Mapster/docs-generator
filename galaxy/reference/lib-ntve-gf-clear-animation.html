# Clear Animation

Grammar — Clear the identifier animation for target
Flags — Action

Clears an animation with the chosen name (Identifier) on an actor.

# Arguments

  • actor — Target
  • string — Identifier

Returns — void

void libNtve_gf_ClearAnimation(
	actor lp_target,
	string lp_identifier,
