# Move Dialog

Grammar — Move dialog to (offsetX, offsetY) relative to anchor of screen
Flags — Native | Action

Moves a dialog to the specified anchor, with the specified offsets.

# Arguments

  • int [ dialog ] — Dialog
  • int [ preset::Anchor ] — Anchor
  • int — OffsetX
  • int — OffsetY

Returns — void

native void DialogSetPosition(
	int dialog,
	int anchor,
	int offsetX,
	int offsetY,
);
