# Order Targeting Relative Points

Grammar — abilCmd targeting relative points near p
Flags — Native | Function

Returns an order targeting relative points near a given point. This is primarily for use in “Issue Order To Unit Group” actions.

# Arguments

  • abilcmd — Ability Command
  • point — Target Point

Returns — order

native order OrderTargetingRelativePoint(
	abilcmd inAbilCmd,
	point inPoint,
);
