# Add Victory Panel Achievement

Grammar — Add achievement achievement to the victory panel
Flags — Native | Action

This will only work with Blizzard maps.

# Arguments

  • string [ gamelink::Achievement ] — Achievement

Returns — void

native void VictoryPanelAddAchievement(
	string inAchievement,
);
