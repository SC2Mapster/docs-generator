# Unit In Region

Grammar — u is in regioin|Region
Flags — Function

Returns true if a unit is in the specified region.

# Arguments

  • unit — Unit
  • region — Region

Returns — bool

bool libNtve_gf_UnitInRegion(unit lp_u, region lp_regioin);
