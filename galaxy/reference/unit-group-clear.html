# Remove All Units From Unit Group

Grammar — Remove all units from g
Flags — Native | Action

Removes all the units in the specified unit group.

# Arguments

  • unitgroup — Unit Group

Returns — void

native void UnitGroupClear(unitgroup inGroup);
