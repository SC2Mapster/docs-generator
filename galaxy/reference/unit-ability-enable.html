# Enable/Disable Ability For Unit

Grammar — enable|Enable/Disable the abil ability for u
Flags — Native | Action

Enables or disables an ability on a unit.

# Arguments

  • unit — Unit
  • string [ gamelink::Abil ] — Ability
  • bool [ preset::Enable_Disable_Option ] — Enable

Returns — void

native void UnitAbilityEnable(
	unit inUnit,
	string inAbil,
	bool inEnable,
);
