# Challenge Mode

Grammar — Set Challenge Mode to onOff for players
Flags — Native | Action | Restricted

This action will only work for Blizzard maps.

# Arguments

  • playergroup — Players
  • bool [ preset::OnOffOption ] — On/Off

Returns — void

native void UISetChallengeMode(
	playergroup players,
	bool challengeMode,
);
