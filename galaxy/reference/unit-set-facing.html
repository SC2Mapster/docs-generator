# Make Unit Face Angle

Grammar — Make u face angle over dur|Time seconds
Flags — Native | Action

Makes a unit face a specified angle.

# Arguments

  • unit — Unit
  • fixed — Angle
  • fixed — Duration

Returns — void

native void UnitSetFacing(
	unit inUnit,
	fixed inFacing,
	fixed inDuration,
);
