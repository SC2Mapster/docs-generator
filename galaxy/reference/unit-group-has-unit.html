# Unit In Unit Group

Grammar — u is in g
Flags — Native | Function

Returns true if the specified unit is part of the specified unit group.

# Arguments

  • unitgroup — Unit Group
  • unit — Unit

Returns — bool

native bool UnitGroupHasUnit(
	unitgroup inGroup,
	unit inUnit,
);
