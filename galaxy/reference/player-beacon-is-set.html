# Beacon Is Set

Flags — Native | Function

Is the specified beacon currently enabled.

# Arguments

  • int — Player
  • int [ preset::AIBeaconType ] — Beacon

Returns — bool

native bool PlayerBeaconIsSet(int inPlayer, int inBeacon);
