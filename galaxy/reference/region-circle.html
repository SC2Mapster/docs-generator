# Convert Circle To Region

Grammar — Region(c, r)
Flags — Native | Function

Returns a circular region from a center point and a radius.

# Arguments

  • point — Center
  • fixed — Radius

Returns — region

native region RegionCircle(point center, fixed radius);

# Related

Category: Region / Creation

  • [F] — region — Convert Circle To Region — ( RegionCircle )
  • [F] — region — Convert Rectangle To Region — ( RegionRect )
  • [F] — void — Add Circle To Region — ( RegionAddCircle )
  • [F] — void — Add Rectangle To Region — ( RegionAddRect )
  • [F] — void — Add Region To Region — ( RegionAddRegion )
