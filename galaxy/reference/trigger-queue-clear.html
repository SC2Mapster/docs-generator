# Clear Action Queue

Grammar — Clear the action queue (activeOption|Retain/Clear/Interrupt the active action)
Flags — Native | Action

Clears all actions currently in the action queue.

# Arguments

  • int [ preset::TriggerQueueActiveOption ] — Active Option

Returns — void

native void TriggerQueueClear(int activeOption);
