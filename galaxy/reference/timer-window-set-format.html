# Set Time Format For Timer Window

Grammar — Set time format for window|Timer Window to format
Flags — Native | Action

Sets the time format for the specified timer.

# Arguments

  • int [ preset::TimerWindow ] — Window
  • text — Format

Returns — void

native void TimerWindowSetFormat(
	int inWindow,
	text inFormat,
);
