# Set Day Length

Grammar — Set day length to length game seconds
Flags — Native | Action

Sets the amount of game time it will take to cycle through 24 hours of time of day.

# Arguments

  • fixed — Length

Returns — void

native void GameTimeOfDaySetLength(fixed inSecs);
