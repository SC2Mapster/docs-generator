# Set Transmission Source Streaming Allowed

Grammar — Set the Streaming Allowed flag for transmissionSource to allowed|allowed.
Flags — Native | Action

If the streaming allowed flag is set to true and this transmission source is a movie, the movie can be streamed if its not local.

# Arguments

  • transmissionsource — Transmission Source
  • bool — Allowed

Returns — void

native void TransmissionSourceSetStreamingAllowed(
	transmissionsource source,
	bool allowed,
);
