# Wave Get

Grammar — Get the wave at index waveIndex for player player
Flags — Native | Function

Returns the wave at the specified index for a player.

# Arguments

  • int — Player
  • int — WaveIndex

Returns — wave

native wave AIWaveGet(int player, int waveName);
