# Player In Cinematic Mode

Grammar — Player player is in cinematic mode
Flags — Function

Returns true if the specified player has cinematic mode turned on.

# Arguments

  • int — Player

Returns — bool

bool libNtve_gf_PlayerInCinematicMode(int lp_player);
