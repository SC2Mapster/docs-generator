# Set Data Conversation Unit

Grammar — Set conversation stateIndex|State to unit
Flags — Native | Action

Sets a conversation state to a unit.

# Arguments

  • string [ convcharacter ] — State Index
  • unit — Unit

Returns — void

native void ConversationDataRegisterUnit(
	string stateIndex,
	unit u,
