# Enable/Disable Revealer

Grammar — enable|Enable/Disable r
Flags — Native | Action

Enables/Disables the specified revealer but does not destroy it.

# Arguments

  • revealer — Revealer
  • bool [ preset::Enable_Disable_Option ] — Enable

Returns — void

native void VisRevealerEnable(revealer r, bool enable);

Category: Visibility / Revealers
