# Remove Global Cutscene Filter

Grammar — Remove global cutscene filter inFilter.
Flags — Native | Action

Removes the filter for all cutscenes. Any nodes with a non-empty filter property that does not match this value will be filtered out of your cutscene.

# Arguments

  • string — Filter

Returns — void

native void CutsceneRemoveGlobalFilter(string inFilter);
