# Pause/Unpause Action Queue

Grammar — pause|Pause/Unpause the action queue
Flags — Native | Action

Pauses or unpauses the Action Queue. Pausing the Action Queue does not pause actions within the trigger that is currently running in the queue, but will prevent actions in subsequent triggers from running until the queue is unpaused.

# Arguments

  • bool [ preset::PauseUnpauseOption ] — Pause

Returns — void

native void TriggerQueuePause(bool pause);

Category: Trigger / Action Queue
