# Set Stock

Grammar — Set stock for player player to make count units of type unitType
Flags — Native | Action

Set stock for a player to make units of the specified unit type.

# Arguments

  • int — Player
  • int — Count
  • string [ gamelink::Unit ] — Unit Type

Returns — void

native void AISetStock(
	int player,
	int count,
	string aliasType,
);
