# Convert Reals To Color

Grammar — Color(r%, g%, b%)
Flags — Native | Function

Converts a real value into a color.

# Arguments

  • fixed — Red
  • fixed — Green
  • fixed — Blue

Returns — color

native color Color(fixed r, fixed g, fixed b);
