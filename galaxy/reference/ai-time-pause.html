# Pause AI Time

Grammar — pause|Pause/Unpause AI time
Flags — Native | Action

Pauses or unpauses the AI time. While paused, the controlling AI will idle. AI owned units will carry out their current actions, and then idle.

# Arguments

  • bool [ preset::PauseUnpauseOption ] — Pause

Returns — void

native void AITimePause(bool inPause);
