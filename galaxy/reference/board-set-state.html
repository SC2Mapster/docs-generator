# Enable/Disable Leaderboard State

Grammar — enable|Enable/Disable state state for board|Leaderboard for players
Flags — Native | Action

Enable/Disable a state for the specified leaderboard.

# Arguments

  • int [ preset::Board ] — Board
  • playergroup — Players
  • int [ preset::BoardState ] — State
  • bool [ preset::Enable_Disable_Option ] — Enable

Returns — void

native void BoardSetState(
	int inBoard,
	playergroup inPlayers,
	int inState,
	bool inVal,
);
