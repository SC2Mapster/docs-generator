# Text Message

Grammar — Display message for players to messageArea|MessageArea area
Flags — Native | Action

Displays a text message in the specified area of the screen.

# Arguments

  • playergroup — Players
  • int [ preset::MessageArea ] — Message Area
  • text — Message

Returns — void

native void UIDisplayMessage(
	playergroup players,
	int messageArea,
	text messageText,
);
