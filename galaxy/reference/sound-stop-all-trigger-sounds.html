# Stop All Trigger Sounds

Grammar — Stop all trigger sounds fade
Flags — Native | Action

Stops all sounds explicitly played through triggers.

# Arguments

  • bool [ preset::SoundFadeOption ] — Fade

Returns — void

native void SoundStopAllTriggerSounds(bool fade);
