# Point With Z Offset

Grammar — p offset by z height
Flags — Function

Returns a point with the X and Y position of the specified point, and Z position of the specified Z.

# Arguments

  • point — Point
  • fixed — Z

Returns — point

point libNtve_gf_PointWithZOffset(point lp_p, fixed lp_z);
